# Intelligent Academic & Placement Tracker (IAPT)

IAPT is a comprehensive full-stack web application designed for engineering students. It acts as a centralized portal to manage academic schedules, track placement applications, and receive AI-driven study recommendations based on performance metrics.

## Tech Stack
- **Frontend**: React.js, Tailwind CSS v3, Vite, Lucide React (Icons)
- **Backend**: Python Flask, RESTful API, JWT Authentication
- **Database**: MySQL
- **Machine Learning**: Scikit-learn (Linear Regression model for study predictions)

## Core Features
- **Interactive Dashboard**: High-level overview of daily classes, active study topics, and overall progress.
- **Timetable Manager**: Add, manage, and view daily class and lab schedules with conflict detection.
- **Placement Tracker**: Log job applications, track interview statuses (Applied, Interview, Offer, Rejected), and compute real-time success rates.
- **AI Study Scheduler**: Personalized daily study time recommendations generated by a trained ML model analyzing past performance, upcoming deadlines, and topic completion.
- **Mock Tests Engine**: Take timed practice exams and review performance scores.
- **Resource Hub**: Access curated study materials, video lectures, and company preparation guides.

##  Local Setup Instructions

### 1. Database Setup
1. Ensure MySQL is running locally.
2. Execute the provided schema to create the necessary database and tables:
   ```bash
   mysql -u root -p < database/schema.sql
   ```
3. Update your local database credentials in `backend/config.py`.

### 2. Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```
3. Install the required Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Start the Flask server:
   ```bash
   python app.py
   ```
   *The backend API will be available at `http://localhost:5000`.*

### 3. Frontend Setup
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install the necessary Node packages:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
   *The frontend application will be available at `http://localhost:5173`.*

##  Machine Learning Engine
The AI Study Plan relies on a Scikit-Learn Linear Regression model. To retrain the model with new synthetic data:
```bash
cd ml
python generate_data.py
python train_model.py
```
This will output a new `model.pkl` file used by the Flask backend for predictions.
